Description:

Outdoor Buddy provides a wealth of useful data for those who love the outdoors by calling to third party APIs via Axios. The frontend is built with React/Redux and Material UI. The backend uses Node, Express and MongoDB..

Features:

  • - Create an account to get the weather information for a specific area with every login.

  • - Enter a zipcode to get data for any U.S. city

  • - Retrieved data is stored in a database temporarily to prevent unnecessary API calls

  • - Information changes depending on relevance given the time of day.
