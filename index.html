About

Hello! I am a tenacious self-taught full-stack developer. I love creating beautiful, useful web applications using React, Redux, Node, Express, MySQL and Mongo DB. I've also used several CSS frameworks including Material UI and React Bootstrap. I've taken formal courses in HTML,CSS, Javascript, Responsive Design, Adobe Photoshop, UX/UI and E-commerce via Austin CC's Web Developer Program. I also completed the online bootcamp The Odin Project through which I have learned to utilize: Git, Github, Linting, Test-Driven Development using Jest and WebPack.

In another life, I was in the medical field and have a B.S. in Economics. I have enjoyed making this career change as I find the field of web-development rewarding, creative, collaborative and filled with new things to learn. I look forward to working with you on your next project!
