Description:

Picturely Prints is an e-commerce site selling photographic prints which integrates the Paypal REST API to allow users to pay for purchase via Paypal, Venmo or Credit/Debit card. The front-end is built with React, Redux and React Bootstrap. The back-end utilizes Node, Express and a MySQL database.

Features:

  • - Paginated display of products with full search functionality

  • - Fully functional shopping cart for managing items and their quantities

  • - An intuitive step-wise checkout system

  • - A customer profile detailing the user's order history and offering them the ability to edit their login details

  • - Product review and rating system

  • - Role-based Authorization/access using JSON web tokens

  • - Administrator's can delete users or adjust their site privileges, create, edit and delete products and manage and track the status of customer orders
